HTML Nedir?

1- HTML, Web sayfalari oluşturmak için kullanilan standart işaretleme dilidir.

2- HTML, Hyper Text Markup Language (Köprü Metni İşaretleme Dili) anlamına gelir

3- HTML, Web sayfaları oluşturmak için kullanılan standart işaretleme dilidir

4- HTML, bir Web sayfasının yapısını tanımlar

5- HTML bir dizi öğeden oluşur

6- HTML öğeleri tarayıcıya içeriğin nasıl görüntüleneceğini söyler

7- HTML elemanları, "bu bir başlıktır", "bu bir paragraftır", "bu bir bağlantıdır" vb. gibi içerik parçalarını etiketler



HTML Temel Örnekler

HTML Belgeleri

Tüm HTML belgeleri bir belge türü bildirimiyle başlamalıdır: "!DOCTYPE html".

HTML belgesinin kendisi ile başlar "html"ve ile biter "/html".

HTML belgesinin görünür kısmı"body"ve arasındadır "/body".

"!DOCTYPE" Beyanı

Bildirim "!DOCTYPE", belge türünü temsil eder ve tarayıcıların web sayfalarını doğru şekilde görüntülemesine yardımcı olur.

Sadece bir kez, sayfanın en üstünde (herhangi bir HTML etiketinden önce) görünmelidir.

Beyanname "!DOCTYPE"büyük/küçük harfe duyarlı değildir.
